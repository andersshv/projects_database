Database med al information om projekter.
En get og post api, hvor man kan tilføje projekter, og aflæse eksisterende projekter.
Et projekt skal enten være kørende eller arkiveret.
Man kan så bede om at få vist alle kørende projekter eller alle arkiverede.
Der skal så være et CLI der kan udskrive i consollen.
En hjemmeside implementation - frontend.
Og en app.
Så man kan tilgå data uanset hvor man står.
Databasen skal have en tabel der hedder: Projekt, som ser sådan ud
<User>
ID: Integer
Brugernavn: String
Password: String
Email: String
Projekter: [<Projekt>]

<Projekt>
ID: Integer
Beskrivelse: String
Oprettelsesdato: String
Gøremål[<Gøremål>]
Afsluttet: Boolean

<Gøremål>
ID: Integer
Beskrivelse: String
Udført: Boolean
